Smart Home Energy Analysis

A data analysis project exploring smart home appliance energy usage using Python (Pandas, NumPy), Matplotlib, and feature engineering.
The project uncovers usage patterns across rooms, appliances, day parts, and weekdays to understand energy consumption behavior.
